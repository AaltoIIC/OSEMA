Setting up new server

(Instructions for TP-Link router)

  • Connect to router
  • Go to http://tplinkwifi.net and log in
  • Select "Wireless clients" from front page
  • There you can find the MAC address and IP address of the server
  • Go to Advanced tab and select Network from the right and then DHCP Server.
  • Reserve address to the server by clicking Add.
  • After you have saved settings go to NAT Forwarding from the right and select Virtual Servers
  • Add new virtual server. Use the previously reserved IP address and port can be freely chosen.
